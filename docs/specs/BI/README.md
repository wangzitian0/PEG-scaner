# BI Specs

Business Intelligence & Data Requirement Documents (DRD) for data coverage, quality, and operational monitoring.

## 职责范围
- 数据覆盖：股票池定义（SP500/KWEB/VGT）、时间跨度
- 数据源配置：yfinance/SEC/Alpha Vantage 对比与选择
- 质量规则：多源校验（≥3来源）、置信度算法
- 运营监控：缺失率、延迟 SLA、异常值检测
- BI Dashboard：爬虫成功率、数据新鲜度、一致性分布

## 不属于此处
- ❌ 数据库 Schema（属于 TRD）
- ❌ 查询实现代码（属于代码仓库）
- ❌ GraphQL 定义（属于 TRD）
