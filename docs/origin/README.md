# Origin (Immutable Inputs)

Store original, user-authored decisions here. AI agents should treat these files as immutable unless the owner explicitly requests a decision update. Examples: `BRN-001.core_infra_ping.md`, `BRN-002.graphql_protocol_decision.md`.

原则上，每个 BRN 都不应该超过 40 行。立项文件，讲清楚核心逻辑即可。
做不做，怎么执行，都不是 BRN 的重点，到 project 去管理。
