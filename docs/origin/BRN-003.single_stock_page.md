# BRN-003: 个股页面（对应 PRD-003）
1) 目标：提供美股单只股票的一站式视图（行情/K线、新闻、F10、因子入口）。
2) 用户：终端投资者/量化用户，移动端优先，Web 兼顾。
3) 场景：查看日/周/月 K 线与成交量，阅读新闻与 F10 关键财务。
4) 数据：行情、新闻、F10 来自多源校验后入库（宁缺勿滥，≥3 来源校验）。
5) 界面：K 线+成交量，新闻列表，F10 子区块（估值、运营、财务估计、指标）。
6) 交互：切换周期（日/周/月），点选新闻/F10 展开详情，加载状态与错误提示。
7) 协议：前后端通过 GraphQL 交互，HTTP/JSON 传输，遵循 TRD-002 定义。
8) 依赖：后端行情/新闻/F10 API、Neo4j 数据、GraphQL Schema（libs/schema/）。
9) 验收：页面加载成功返回 200，K 线/新闻/F10 块可展示；回归用 `regression:web-e2e` 通过。
